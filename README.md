# Design Patterns em C#

Este repositório contém exemplos de implementação dos principais padrões de projeto (Design Patterns) utilizando a linguagem C#.

## Estrutura do Projeto
- `Singleton/`: Implementação do padrão Singleton
- `Main.cs`: Exemplo de uso do Singleton
- `Singleton.cs`: Implementação do Singleton

## Como executar
1. Certifique-se de ter o .NET instalado em sua máquina.
2. Execute o comando abaixo para compilar o projeto:
   ```cmd
   dotnet build
   ```
3. Para rodar o projeto:
   ```cmd
   dotnet run
   ```

## Sobre Design Patterns
Design Patterns são soluções reutilizáveis para problemas comuns de desenvolvimento de software. Eles ajudam a criar sistemas mais organizados, flexíveis e fáceis de manter.
